maproulette helps you find interesting places on the way.
it works by taking a start and endpoint and delivers a random path
between the two using waypoints provided by foursquare.
you can find the source on github.

maproulette was developed using google's appengine, foursquare, and yelp
geocoding courtesy mapquest's open api
